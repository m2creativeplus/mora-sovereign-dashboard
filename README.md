# 🇸🇱 MORA Sovereign Intelligence Dashboard

> Institutional Governance. Financial Sovereignty. Digital Excellence.

The **Somaliland Ministry of Religious Affairs (MORA) Sovereign Intelligence Dashboard** is a high-performance digital platform designed to centralize the management of national endowments (Waqf), Halal export traceability, and Sharia-compliant financial monitoring.

## 🚀 Core Features

-   **GIS Waqf Registry:** Real-time geospatial tracking and recovery of state-owned religious endowments.
-   **Halal Export Traceability:** Blockchain-verified monitoring of livestock exports along the Berbera Corridor.
-   **National Zakat Gateway:** Transparent, leak-proof digitalization of religious social finance.
-   **Kaltirsi Integration:** Unified Somali-Islamic calendar orchestration for institutional scheduling.
-   **Islamic Finance Audit:** Substantive compliance monitoring for the national banking sector.

## 🏗️ Tech Stack

-   **Frontend:** Next.js 15+ (App Router), TypeScript, Tailwind CSS
-   **Animations:** Framer Motion
-   **Backend:** Convex (Real-time data engine)
-   **Auth:** Clerk (Sovereign Identity)
-   **Aesthetics:** Void & Gold (#D4AF37), Glassmorphism, Premium Dark Mode.
